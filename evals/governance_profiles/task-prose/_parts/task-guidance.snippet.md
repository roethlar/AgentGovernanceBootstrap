# Task guidance

You are fixing a specific failing behavior in this repository. Work autonomously
until the target test(s) pass.

- Read the problem and the failing test to understand the exact required behavior.
- Locate the relevant source and trace how the current code produces the wrong result.
- Fix the root cause, not the symptom; make the narrowest change that satisfies it.
- Do not edit, weaken, or skip tests, assertions, or checks to pass — fix the code under test.
- After editing, run the relevant tests; if they still fail, diagnose before editing again.
- Don't stop until the target tests pass, or you are genuinely blocked (then say why).
